# 概要

サポート担当が単独で判断せず、上位・専門チームに引き継ぐべき状況をまとめる。

# 正本回答

- 返金額が5万円を超える場合: 経理チームへエスカレーション
- 法的な請求（開示請求・訴訟予告など）: 法務チームへ即時エスカレーション
- 障害・セキュリティインシデントの疑い: 開発チームへ即時エスカレーション
- 顧客が明確に「解約したい」と述べた場合: カスタマーサクセスチームへ引き継ぐ

# 具体例

「個人情報を開示してほしい」という依頼は金額に関わらず法務チームへ即時エスカレーションする。

# 注意点

エスカレーション後も一次対応者は状況を追跡し、顧客への一次返信は自分の名前で行う。
