# 概要

問い合わせの受付からクローズまでの標準的な対応フローを定める。

# 正本回答

1. 受付後1時間以内に一次返信する（テンプレート「受付確認」を使用）
2. エスカレーション基準に該当するか確認する
3. 該当しない場合は、返金ポリシー・料金FAQ等の正式ナレッジを根拠に回答する
4. 顧客からの返信が2営業日ない場合はフォローアップを1回送る
5. 解決したら「解決済み」に変更し、対応内容を1行で記録する

# 具体例

一次返信は「お問い合わせありがとうございます。担当が確認し、24時間以内にご案内します。」
のようなテンプレートを使う。

# アンチパターン

- 根拠となるverified noteを示さずに独自判断で返金額や特例を案内しない
- エスカレーション基準に該当する内容を自己判断で処理しない
